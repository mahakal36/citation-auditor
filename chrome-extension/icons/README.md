# Extension Icons

You need to create icon files for the Chrome extension:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

You can use any image editing tool to create these, or use an online icon generator.

For a quick solution, you can use a simple colored square or letter icon.

## Quick Icon Creation

1. Go to https://favicon.io/favicon-generator/
2. Enter a letter like "C" for Citation
3. Choose colors (purple #8b5cf6 works well)
4. Download and extract the icons
5. Rename/resize to match the required sizes above
